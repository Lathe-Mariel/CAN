KiCADで作ったPCBデータです．  
CANバスをチェックするためのアナライザをESP32-DevKitCとTJA1441AT（NXP）で作ります．  

CANバスとは，RJ45で接続します．中心寄りの2芯を使っています．  
RJ45を2個搭載できるようになっているので，数珠つなぎにできます．従ってこのまま，センサノードとして利用しやすくなっています．  
ESP32_DevKitCの全ピンが引き出されており，重ねる基板を作りやすくなっています．
